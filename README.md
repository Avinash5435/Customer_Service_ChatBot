# Generative AI Q&A System using Google Gemini and LangChain

This is an end-to-end LLM project built using **Google Gemini**, **LangChain**, and **Streamlit** for an e-learning company called **Nullclass**.

Nullclass offers data-related courses and virtual internships. They receive thousands of learner questions through **Discord** and **email**. This project helps automate that support process by building an AI-powered **Question and Answer system** that can answer student queries in seconds.

## Features

- Built using **Google Gemini LLM**
- Uses **LangChain** for orchestration
- Streamlit-based user interface
- Uses a real **CSV FAQ dataset**
- Creates a **FAISS vector database** for semantic search
- Helps reduce the workload of human support staff
- Provides quick and accurate answers for students

## Project Highlights

- Uses a real FAQ CSV file currently used by Nullclass staff
- Supports students with instant answers
- Reduces manual effort for customer support teams
- Easy-to-use Streamlit web interface
- Knowledge base is created from FAQs and stored locally
